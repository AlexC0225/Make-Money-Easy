# Make Money Easy

這是一個用來管理台股觀察名單、整理持倉、查看策略訊號，以及做回測的工具。

這份說明是給使用者看的，只講你會用到的內容：
- 資料什麼時候更新
- 股票池怎麼選
- 各策略在看什麼
- 每個頁面怎麼使用

## 資料更新時間

系統時間以台灣時間為準。

### 每天固定更新

#### 早上 06:00
- 更新股票基本資料
- 例如股票代碼、名稱、產業分類

#### 早上 09:30
- 執行你在 Dashboard 設定好的自動化策略
- 如果條件成立，系統會依照你的設定決定買進、賣出或跳過

#### 下午 14:10
- 更新收盤後的歷史資料
- 讓隔天看盤、回測、訊號判斷有最新資料可用

### 你也可以手動更新

在使用過程中，不一定要等排程。

- 在 `Setup` 可以手動更新股票資料
- 在 `Setup` 可以手動同步某些股票的歷史資料
- 在 `Market` 調整 watchlist 之後，也會影響後續同步範圍

## 股票池怎麼選

系統主要有兩種股票池概念。

### 1. 你自己指定的股票
- 只要你手動輸入股票代碼，就只會針對那些股票處理
- 適合你已經有明確觀察名單的時候使用

### 2. 系統預設股票池
- 系統會自動整理一批預設股票
- 也會把你加入 watchlist 的股票一起納入

### 預設股票池目前偏重的產業
- 半導體
- 電腦及週邊設備
- 電子零組件
- 通信網路
- 其他電子
- 金融保險
- 資訊服務
- 數位雲端

### 系統還會做流動性篩選
- 近期成交金額太小的股票，會被排除
- 目的很簡單：避免把太冷門、太難交易的股票放進主要觀察池

### watchlist 的作用
- 你加入 watchlist 的股票，會變成系統優先關注的標的
- 就算不在預設產業內，只要你有加入 watchlist，系統也可以一起追蹤

## 策略說明

目前系統提供 4 種策略方向。

### 趨勢 + 四大買賣點
- 適合想看趨勢方向的人
- 主要看短中期均線、價格位置、量能是否配合
- 當趨勢轉強時偏向買進
- 當趨勢轉弱或跌破關鍵均線時偏向賣出

### RSI 短線反彈
- 偏向找大方向仍然向上，但短線明顯超跌的股票
- 重點是找短線反彈機會
- 持有時間通常不長
- 比較適合節奏快、想做短波段的人

### 日線開盤動能
- 偏向找價格、趨勢、量能都同步轉強的股票
- 會搭配風險控制，不是只看買點
- 除了進場條件，也會看停損、持有天數、動能是否變弱
- 適合想做相對穩健動能策略的人

### 動能突破
- 偏向找突破整理區、成交量放大的強勢股
- 會看趨勢、突破、量能、強弱指標
- 進場後也會持續檢查是否轉弱
- 適合喜歡追蹤強勢股的人

## 功能怎麼用

### Login
- 用既有帳號登入
- 登入後就可以進入你的 workspace

### Setup

這是你開始使用前最重要的一頁。

你可以在這裡：
- 建立或更新你的 workspace
- 設定初始資金與可用現金
- 匯入目前持倉
- 手動更新股票資料
- 手動同步歷史資料

建議第一次使用時這樣做：
1. 建立 workspace
2. 設定資金
3. 匯入持倉
4. 更新股票資料
5. 同步歷史資料

### Dashboard

這裡是你的主畫面。

你可以在這裡：
- 看資產總覽
- 看目前持倉
- 看 watchlist
- 看股票報價與圖表
- 設定自動化策略

重點提醒：
- 你在這裡做的是「設定規則」
- 真正的自動執行會在每天早上 09:30 跑

### Market

這裡主要是管理你的 watchlist。

你可以在這裡：
- 新增想追蹤的股票
- 移除不想追蹤的股票
- 檢查目前系統主要會關注哪些股票

如果你想讓某些股票被系統持續追蹤，這頁很重要。

### Backtests

這裡是用來驗證策略的地方。

你可以在這裡：
- 選策略
- 選股票
- 選日期區間
- 設定資金
- 跑回測
- 看結果與交易紀錄

適合什麼時候用：
- 想先看策略過去表現
- 想比較不同股票、不同區間、不同策略的差異

### Signals

這裡是快速看訊號的地方。

你可以在這裡：
- 看最近有哪些股票出現買進、賣出或觀望訊號
- 看訊號原因
- 看最近交易紀錄

如果你想快速知道系統最近在看什麼，先看這頁就對了。

## 建議使用順序

如果你是第一次使用，建議照這個順序：

1. 先到 `Setup` 完成資料建立
2. 到 `Market` 整理 watchlist
3. 到 `Dashboard` 設定策略
4. 到 `Backtests` 驗證策略想法
5. 到 `Signals` 查看最新訊號
